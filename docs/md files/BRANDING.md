# Branding Guide

## Product Name

- Primary: `MCL Studio`
- Expanded: `Markdown Component Layout Studio`

## Brand Positioning

MCL Studio should feel like a developer-grade design and parsing tool:

- precise
- modern
- technical
- editorial

It should align more closely with:

- Vercel
- Linear
- Raycast
- Framer

It should avoid:

- mascot branding
- gaming aesthetics
- academic-project visuals

## Logo System

Files:

- `public/logo.svg`
- `public/logo-dark.svg`
- `public/favicon.svg`

Visual idea:

- angle brackets communicate parsing and language tooling
- the central framed block suggests a layout canvas
- branching nodes suggest the AST and developer inspection workflow

## Usage

### Light or neutral backgrounds

Use:

- `public/logo.svg`

### Dark surfaces or inverted usage

Use:

- `public/logo-dark.svg`

### Browser and tab icon

Use:

- `public/favicon.svg`
- generated `public/favicon.ico`
